# Product Overview

Zodi is a premium astrology application that provides personalized daily horoscopes, compatibility analysis, and detailed astrological insights powered by Google's Gemini AI.

## Core Features

- Daily horoscope readings with metrics (love, money, health, career)
- Zodiac compatibility matching between signs
- Detailed astrological analysis by category
- Premium/freemium model with upgrade flow
- Dark/light theme support
- User authentication and profile management

## Target Platform

- Primary: Web application (React/Vite)
- Secondary: Android native app (Kotlin/Jetpack Compose)

## User Experience

The app uses a conversational, casual tone ("Zodi") that delivers honest, sometimes blunt astrological insights. Content is in Turkish, targeting Turkish-speaking users with a modern, relatable approach to astrology.

## Monetization

Freemium model with premium features unlocked through in-app purchase flow.
